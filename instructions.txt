*After installing Emscripten and wasm 
*Add all the data structures ADT's in the path and run the following instructions in cmd

1) Open the right terminal in VS Code
2) Enable running scripts for this terminal session (temporary, safe)

    Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

3)Activate Emscripten SDK for this terminal session

    cd C:\Users\bilal\emsdk
    .\emsdk activate latest
    .\emsdk_env.ps1

4)Confirm emcc is available

    emcc --version
    where.exe emcc
    python --version

5) Put a build script in your project and run it by this command (recommended)

    .\build\build.bat
